# GDB Store — Jornada completa

Protótipo visual em HTML único para marketplace de produtos avulsos do Grupo Dom Bosco.

## Como publicar no GitHub Pages

1. Substitua o `index.html` antigo por este novo `index.html`.
2. Faça commit na branch `main`.
3. Aguarde o GitHub Pages atualizar.

Tudo está no próprio `index.html`: layout, CSS e interações.
